CPSC 310 HTML Example

This is an example HTML file. The syntax is fairly intuitive, the <html> tag at the top is closed by the </html> tag at the bottom of the document. Further tags are then nested inside these tags recursively to structure the document. There are many tags such as:

  1. <body> - which stores the actual document contents.
  2. <h1> - which indicates that the content is a header.
  3. <p> - indicates that the content is a paragraph.
  4. <em> - indicates that the content should be emphasized
  5. <img> - places an image in the document

The CSS File included at the top drastically changes the visual aspects of this page, without changing the structure of the actual document.

The Javascript file included at the top of the page lets us add more complicated functionality.
